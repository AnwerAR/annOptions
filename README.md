# annOptions
A flexible plugin to create WordPress admin panels on the go.
> *Caution: Still Under Development & not yet ready*
## Basic Usage
```
add_action ( 'ao_options', function ( $options ) {

  $options->setPage(
    'page_title' => __( 'Page Title' ),
    'menu_title' => __( 'Page Title' ),
    '....'       => '...'
  );
});
```
